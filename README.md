## 📌 About The Repository
This rigorous sprint is designed to supercharge coding skills by tackling diverse LeetCode problems and mastering essential algorithms. While the initial structured phase covers 100 days, this repository serves as a continuous technical diary of my ongoing progress, showcasing consistent problem-solving and a strong foundation in core Computer Science concepts.

## 💻 Tech Stack
* **Language:** Java (JDK 17+)
* **Platform:** LeetCode
* **Concepts:** Data Structures and Algorithms (DSA)

## 📚 Core Topics & Techniques Covered
This repository focuses on genuine, high-frequency interview topics covered in the edSlash playlist and standard technical assessments:
* **Core Data Structures:** Arrays (1D & 2D), Strings, Linked Lists, Stacks, and Queues.
* **Trees:** Binary Trees and Binary Search Trees (BST), including traversals and root-to-leaf paths.
* **Hashing:** Extensive use of HashMaps and HashSets for frequency counting and fast lookups.
* **Problem-Solving Patterns:** * Two Pointers
  * Sliding Window
  * Prefix Sum (e.g., Pivot Index, Maximum Subarray)
* **Algorithmic Basics:** Binary Search, Sorting algorithms, Recursion, and basic Dynamic Programming.
